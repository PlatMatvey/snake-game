# Snake Game

A simple Snake game written in Python using the Pygame library.

## Features

* WASD controls
* Score counter
* Random food spawning
* Collision with walls and the snake's body

## Requirements

* Python 3
* Pygame

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

```bash
python src/main.py
```
